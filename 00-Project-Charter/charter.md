# Project Name
BlackPhoenix SOC Platform (BPSP)

# Type
Multi-Region Cloud Security Operations Center

# Platform
Microsoft Azure

# Duration
8-10 Days

# Problem Statement
Most student SOC labs are single-machine environments and do not represent enterprise security operations.

This project builds an enterprise-style SOC entirely on Azure using multiple regions, distributed networks, centralized monitoring, attack simulation, and automated incident response.

# Objectives
* Build multi-region Azure infrastructure
* Deploy Active Directory
* Deploy Windows and Linux endpoints
* Deploy SIEM and SOAR
* Simulate attacks
* Engineer detections
* Perform threat hunting
* Conduct incident response

# Success Criteria
✓ All 8 VMs deployed
✓ Multi-region communication works
✓ Logs reach SIEM
✓ Alerts generated
✓ Cases created
✓ Playbooks executed
✓ Incident reports completed
